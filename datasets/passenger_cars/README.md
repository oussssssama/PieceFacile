# Passenger Cars Reference Dataset

## Overview

This dataset is the official reference for Passenger Cars used by the PieceFacile platform.

It serves as the source of truth for vehicle selection in Version 1 of the application.

## Status

Approved

## Version

V1.0

## Market

Algeria (DZ)

## Purpose

This dataset is used by:

- Buyer Request Creation
- Seller Vehicle Specialization
- Smart Search
- Autocomplete
- Marketplace Filtering
- Analytics

## Database Mapping

The dataset populates the following database tables:

- vehicle_categories
- brands
- models

## Scope (Version 1)

Included:

- Vehicle Category
- Brands
- Models

Deferred to Version 2:

- Families
- Generations
- Engines
- Vehicle Compatibility

## Notes

This dataset follows the PieceFacile V1 reference architecture.

Passenger Cars are represented using the following hierarchy:

Vehicle Category
→ Brand
→ Model
