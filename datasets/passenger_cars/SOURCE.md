# Passenger Cars Source

## Official Source

PieceFacile_Passenger_Cars_V1.xlsx

## Purpose

This Excel workbook is the authoritative source for the Passenger Cars dataset.

The documentation in this directory summarizes the dataset.

The database seed files are generated from the Excel workbook.

## Version 1 Scope

Only the following entities are imported:

- Vehicle Category
- Brands
- Models

The following entities are intentionally excluded from Version 1:

- Families
- Generations
- Engines
- OEM References
- VIN Compatibility

## Notes

Do not edit generated SQL files manually.

Any change to brands or models must first be applied to the source workbook.
