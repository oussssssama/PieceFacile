# MATCHING STRUCTURE

LCV Matching uses:

Vehicle Category
+
Brand
+
Model
+
Part Category

Example:

LCV
→ Toyota
→ Hilux
→ Suspension

LCV
→ Renault
→ Kangoo
→ Engine Parts

LCV
→ Fiat Professional
→ Doblo
→ Body Parts

This structure is used by:

* Smart Search
* Seller Specializations
* Request Visibility
* Matching Engine
* Analytics
